编辑器核心引擎

实现 WYSIWYG Markdown 编辑，底层用什么技术

A

Milkdown（推荐）

基于 ProseMirror 的 Markdown WYSIWYG 编辑器框架。插件化架构，开箱即用的 Markdown 支持，主题可定制。轻量且现代。

B

Tiptap

同样基于 ProseMirror，社区成熟，生态丰富。需要自己配置 Markdown 序列化/反序列化，但灵活度更高。

C

Vditor

国产开源 Markdown 编辑器，内置所见即所得模式、即时渲染模式。中文友好，功能完善，开箱即用。体积较大。

D

自研 contenteditable

基于 contenteditable + Markdown 解析器自己实现。完全掌控，但开发量巨大，坑多。不推荐 MVP。
